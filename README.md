# 🦊 八重神子工作陪伴

让鸣神大社宫司、八重堂总编八重神子在 VS Code 中陪你写代码、审稿与找灵感。

## 功能

- 🦊 优雅毒舌风格对话
- 🍅 番茄钟专注与休憩提醒
- 💬 关键词智能回复
- 🔍 审稿式代码检查
- 📊 每日统计：番茄钟 / 对话 / 保存
- 📚 独有能力：总编点评 / 小说反转 / 影的彩蛋
- ✨ 打字机效果与侧边栏聊天面板

## 命令

- `神子：今日开卷`
- `神子：开始番茄钟`
- `神子：结束番茄钟`
- `神子：给我一点建议`
- `神子：审稿式检查代码`
- `神子：今日收卷`
- `神子：总编点评`
- `神子：来个小说反转`
- `神子：聊聊影`

## 运行

安装依赖后编译：

- `npm install`
- `npm run compile`

然后在 VS Code 中按 `F5` 启动调试。
